# 📊 Customer Purchase Behavior Analysis

## 🔍 Overview
This project focuses on analyzing customer purchase behavior using transactional data. The goal is to extract meaningful insights into customer preferences, spending patterns, and engagement to support data-driven business decisions.

---

## 📁 Dataset
The dataset contains ~3900 records with 18 features, including:
- Customer demographics (Age, Gender, Location)
- Purchase details (Category, Item, Amount, Season)
- Behavioral data (Discount usage, Frequency, Ratings, Subscription status)

---

## 🛠️ Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **SQL** (PostgreSQL / MySQL)
- **Power BI**
- **Jupyter Notebook**
- **Gamma (for presentation)**

---

## ⚙️ Project Workflow

### 1. Data Loading & Cleaning
- Loaded dataset using Pandas  
- Handled missing values  
- Standardized column names  
- Performed data consistency checks  

### 2. Exploratory Data Analysis (EDA)
- Analyzed purchase distribution  
- Identified trends across categories  
- Compared customer segments  
- Visualized insights using charts  

### 3. SQL Analysis
- Revenue analysis by gender  
- High-value customer identification  
- Product performance analysis  
- Subscription vs non-subscription behavior  
- Customer segmentation  

### 4. Dashboard (Power BI)
- Built interactive dashboard with KPIs:
  - Total Customers  
  - Average Purchase Value  
  - Revenue by Category  
  - Sales by Age Group  

---

## 📊 Key Insights
- Majority of customers fall into the **loyal segment**  
- Younger customers contribute the highest revenue  
- Discounts influence purchasing behavior significantly  
- Non-subscribers generate higher overall revenue
